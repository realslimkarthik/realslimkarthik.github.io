Projects

Fantasy Tweet League

A compelling real time fantasy sports game where scores of players are calculated based on Twitter mentions.

  • Built using Node.js using the Express.js framework.
  • Passport.js used for Twitter authentication.
  • Socket.io used for the real time game itself.
  • jQuery used for the frontend.
  • MongoDB used as the database.

Browse With Me

An application that allows users to browse web pages as a group in real time and discuss it with the built in chat room.

  • Node.js backend
  • Socket.io used for real time communication between multiple users.
  • Lightweight mobile equivalent of jQuery, Zepto.js used for fronted.

JS-Terminal

A Unix-like terminal implemented in HTML5/CSS3/JavaScript which consists of some basic command line commands such as ls, cat, etc. It uses the HTML5 FileSystem API.

GrapeVine

A review application that is used to give reviews about advertisements. Also performs sentiment analysis on recent Tweets about the product or company being advertised to give the user a better picture of what the advertisement really means.

  • Python-Flask backend.
  • MySQL database server to store reviews.
  • Twython for obtaining Tweets to perform sentiment analysis.
  • jQuery for frontend interaction.

PL Results Scraper

A Scrapy spider that scrapes every result of the Barclay's Premier League from its inception in 1992 and stores it in a MongoDB collection. Uses PyMongo for the interaction with the MongoDB database. I've planned a visualization with all this data which I'm currently working on.
